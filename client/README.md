# tBench client

Клиентская часть веб-приложения

## Установка и запуск

### Сначала установить зависимости: `npm install`

### Для запуска использовать `npm run`:
- `dev` - запуск в режиме разработчика
- `build` - сборка проекта
- `start` - запуск собранного проекта

## Навигация

- `components` - компоненты приложения (кнопки, инпуты, формы)
- `helpers` - вспомогательные функции
- `hooks` - кастомыне хуки
- `interfaces` - соответственно, интерфейсы
- `locales` - библиотека переведённых на 5 языков надписей
- `page_components` - компоненты страниц
